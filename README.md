# HireNow

HireNow is a web-based platform designed to connect hirers, contractors, and laborers by simplifying job posting, discovery, and hiring workflows.

## Overview

The goal of HireNow is to reduce friction in informal and short-term hiring by providing a centralized platform where:
- Hirers can post jobs and manage hiring
- Contractors can find and manage work opportunities
- Laborers can access job listings more efficiently

This project is currently under development and serves as a functional prototype.

## Features

- User authentication
- Job posting and browsing
- Role-based access (hirers, contractors, laborers)
- Subscription-based access tiers
- Payment handling via third-party providers

## Tech Stack

- Frontend: React / Next.js
- Backend: Node.js
- Database: PostgreSQL
- ORM: Prisma
- Deployment: Vercel
- Payments: Stripe
